<h1>
What is AutoJDK?
</h1>
<p>
AutoJDK is a tool made in Shell Script to facilitate the installation of the OpenJDK package in several Linux distributions.
</p>
<p>
It is available in 2 different versions. The Shell version and Dialog version.
</p>
